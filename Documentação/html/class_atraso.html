void Atraso::getAtrasoCin ( const char *  mensagem  ) 

Método que pede ao utilizador para introduzir um atraso no formato <DD:HH:mm> ou <HH:mm>.

Enquanto não for introduzido um Atraso no formato pedido continua a pedir ao utilizador para introduzir um Atraso. O objecto que chamou o método fica igual aos dados que o utilizador introduziu.

Parâmetros:
mensagem Mensagem a ser mostrado ao utilizador antes de pedir o input.
