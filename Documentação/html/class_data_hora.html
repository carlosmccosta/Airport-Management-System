void DataHora::getDataHoraCin ( const char *  mensagem  ) 

Método que pede ao utilizador para introduzir uma data no formato <AAAA/MM/DD HH:mm>.

Enquanto não for introduzido uma data no formato pedido continua a pedir ao utilizador para introduzir a data. O objecto que chamou o método fica igual aos dados que o utilizador introduziu.

Parâmetros:
mensagem Mensagem a ser mostrado ao utilizador antes de pedir o input.
